スマホ・PCでPDFに署名する “全部入り” ガイド

機種タブをタップすると、そのやり方だけが表示されます。標準機能オンリー、最短で確実に。

iPhoneで署名（標準マークアップ）

追加アプリ不要。メール/ファイル/写真から直接編集できます。

PDFを開く： 添付メールならプレビューを開き マークアップ（ペン先）をタップ。
ファイルAppならPDFをタップ → 右上 マークアップ。
テキスト署名： 右下「＋」→ テキスト → 氏名・日付・役職などを入力。aA でフォント/サイズ/色を調整、ドラッグで位置合わせ。
手書き署名： 下部ツールからペン → 指やApple Pencilでサイン。二本指ピンチで拡大→精密に。
複数ページ： 左上の縮小表示（サムネイル）でページ移動 → 各ページに同様に入力。
保存/共有： 右上 完了 → 共有。ファイルに保存／メール返信／AirDrop など用途に応じて選択。

見た目を整えるコツ

  • 署名欄の罫線に合わせたい時は、フォントを等幅寄りに、色は濃いグレーにすると紙でも自然。
  • 位置がズレる場合は拡大して微調整。長押しでテキスト枠を掴むと狙いやすい。

改変されにくくする（簡易フラット化）

  1. 共有 → プリント を選択。
  2. プレビューを二本指でピンチアウトして全画面化。
  3. 左上共有 → ファイルに保存。
    多くの注釈が画像化され、動かされにくくなります。

トラブル対処

  • 署名が薄い → ペンの太さ/不透明度を上げる。背景が暗いPDFは白ペンにすると読める。
  • メールに添付できないほど重い → 共有 → プリント → PDF化で軽くなることが多い。

Androidで署名（Googleドライブ注釈）

端末差があるため、ドライブアプリが最も安定。ローカル運用が必要なら後述の印刷/手書きも検討。

PDFを開く： Googleドライブにアップロード → タップして表示。
注釈で入力： 画面下 注釈（鉛筆）→ テキスト を配置 → 氏名・日付を入力。
保存： 右上 保存。元ファイルを残したい時は「コピーを保存」。

仕上げのコツ

  • テキストの色は濃いグレーが紙でも自然。
  • 位置合わせはズームしてからドラッグ。必要に応じて改行で行幅を合わせる。

セキュリティ注意

機密文書はクラウドに載せたくない場合があります。そうした案件はローカル編集が可能なPC（Mac/Windows）か、印刷/手書きルートが安全です。

トラブル対処

  • 注釈が相手側で動く → 一度PCで開き「印刷→PDF」で固定化してから送付。
  • ファイル容量が大きい → 白黒モードに変換して保存すると軽量化しやすい。

Macで署名（プレビュー）

PDFを開く： ダブルクリック（プレビュー）。
テキスト署名： ツール → 注釈 → テキスト。フォント/サイズ/色はマークアップツールバーで調整。
手書き署名： マークアップツールバー → 署名 → トラックパッド/カメラで作成 → 挿入。
保存： 通常保存、または ファイル → プリント → PDFとして保存 で簡易フラット化。

きれいに見せる小ワザ

  • 署名欄のベースラインに合わせ、行頭に全角スペースを足して微調整。
  • 色は#111～#333相当の濃いグレーが自然。黒だと“貼った感”が出やすい。

パスワード付与（任意）

  1. ファイル → 書き出す を開く。
  2. フォーマット: PDF → 暗号化にチェック → パスワード設定。

トラブル対処

  • 文字がにじむ → テキストサイズを一段上げ、PDF保存時の解像度を下げすぎない。
  • 印刷位置がズレる → 用紙サイズ（A4/Letter）と余白設定をPDF側に合わせる。

Windowsで署名（Microsoft Edge）

PDFを開く： 右クリック → プログラムから開く → Microsoft Edge。
テキスト署名： 上部 テキストの追加 → 氏名・日付を入力し、ドラッグで配置。
手書き風： 描画 を選び、マウス/タッチでサイン。太さ・透明度を調整。
保存/固定化： フロッピーの保存、または 印刷 → Microsoft Print to PDF で固定保存。

よくある落とし穴

  • フォント差で見え方が変わる → 受け手の環境差を避けたい場合は必ず「印刷→PDF」で固定。
  • 複数ページで同じ署名を入れたい → 1ページに作ってからコピー＆ペースト→位置合わせ。

印刷/手書き（コンビニ）

「手書き必須」や紙原本回収の指定がある相手に一番確実。

データ準備： USB保存 or 専用アプリ（例：セブン=netprint、ファミマ/ローソン=PrintSmash）にアップロード。
印刷： マルチコピー機で 文書プリント。片面/両面、拡大縮小、向きを確認。
記入： ボールペンで署名・日付。訂正は二重線＋訂正印の指示があるか事前確認。
スキャン： スキャン → PDF、解像度は 300dpi 目安、白黒/文字モード推奨。USB/スマホ/メールへ保存。

仕上がりを良くする

  • 曲がりやすい → 原稿台のガイドに合わせ、縁をしっかり当てる。
  • 暗い/重い → 白黒（文字）モード、300dpiで十分。背景ムラが減り容量も軽くなる。

ファイル名ルール（おすすめ）

署名済_YYYYMMDD_相手名_v1.pdf のように日付・相手・版数を入れるとミスが減ります。

送付方法（最終チェック）

件名・本文： 件名に 署名済 を明記。本文で「締切・差し戻し方法・追伸」を簡潔に。
セキュリティ： パスワード別送（SMS/別メール）。Macは 書き出す→暗号化 可。暗号化ZIPも選択肢。
最終確認： 署名／日付／イニシャル／ページ抜け／回転ズレ／容量（10MB目安）をチェック。
